### Core Functionality
- `verifyAgentInference(proof, publicInputs, proofHash)`  
  → Any medical AI Agent can submit zk-SNARK proof after each inference for on-chain verification
- `isInferenceVerified(proofHash)`  
  → Hospitals, regulators, and auditors can instantly verify every Agent decision
- Designed for nuclear medicine (Ga-68 PSMA PET/CT) with full HIPAA privacy compliance
